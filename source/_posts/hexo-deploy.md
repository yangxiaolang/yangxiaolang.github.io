# Hexo安装环境
  * Node.js
  * Git

# Hexo安装并使用
  1. 全局安装Hexo命令行  

     > npm install -g hexo-cli  

  2. 初始化项目  

     > \$ hexo init <folder>
     > \$ cd <folder>
     > \$ npm install
  
  3. 需要用到的命令
     > \$ hexo s
     > \$ hexo g
     > \$ hexo deploy
